# Itero Testbed

Itero Testbed is a staging environment for browser extensions. Test your browser extensions instantly without dragging and dropping zip files or waiting for browser stores to approve your updates.

## Prerequisites

You must have an account on [Itero](https://itero.plasmo.com). If you don't have an account, you can sign up [here](https://itero.plasmo.com).

## Github Integration

Itero Testbed can be integrated with Github. This allows you to automatically deploy your browser extensions to Itero Testbed when you push to your GitHub repository.
